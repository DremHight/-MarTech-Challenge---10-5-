## 项目结构

### Main分支（Default）
该分支为主要的开发分支，与项目有关的说明和代码文件可放置于此，在仓库被访问时默认展示该分支。
```
-|
--LICENSE   开源协议文件，默认为MIT开源协议。
--README.md 项目说明文件，可使用Markdowm编辑器进行编辑。
--requirements.txt Python项目依赖列表
```
